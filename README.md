# Jestest1

A Vite-based project with Jest testing set up for ECMAScript Modules (ESM).

## Features

- **Vite**: Super fast dev server and bundler.
- **Jest**: Unit testing framework configured for ESM.
- **Stack Implementation**: A sample Stack class in `src/stack.js`.
- **Counter Implementation**: A sample counter in `src/counter.js`.

## Getting Started

### Installation

```bash
npm install
```

### Running Locally

```bash
npm run dev
```

### Running Tests

```bash
npm test
```

## Project Structure

- `src/`: Source code.
  - `stack.js`: Stack class.
  - `counter.js`: Counter logic.
  - `main.js`: Main entry point.
- `test/`: Unit tests.
  - `stack.test.js`: Tests for the Stack class.
  - `counter.test.js`: DOM tests for the counter.
- `public/`: Static assets.
